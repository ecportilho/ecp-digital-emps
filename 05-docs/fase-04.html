Resultados A/B Testing (30 dias)

EXP-001WINNER — Ship Variant
Onboarding Guiado
Hipótese: Se oferecermos um tour guiado após abertura de conta, então a taxa de ativação aumenta.
Controle: Self-discovery (sem tour) • Variante: Tour guiado de 5 passos
+12pp activation rate (Controle: 31% → Variante: 43%) • p = 0.0003 ✅
Tour guiado reduz time-to-first-action de 4.2h para 1.1h. KR-01 superado com a variante.
EXP-002Sem diferença significativa
Quick Actions Position
Hipótese: Se movermos quick actions para painel lateral, então toggles PF/PJ aumentam.
Controle: Topo do dashboard • Variante: Painel lateral
+4.9% toggles/sessão (não significativo, p = 0.287) • Manter controle
Posição dos quick actions não influencia significativamente o toggle PF/PJ. Investigar outras alavancas para KR-03.
EXP-003WINNER — Ship Variant
Boleto Smart Defaults
Hipótese: Se pré-preenchermos juros/multa com defaults inteligentes, então mais empresas emitem boletos.
Controle: Campos vazios • Variante: Pré-preenchido (1% juros, 2% multa)
+18.1% boletos/empresa/mês (Controle: 1.8 → Variante: 2.4) • p = 0.0021 ✅
Defaults inteligentes reduzem fricção cognitiva e confirmam princípio "Automação vence configuração". KR-02 superado.
